# aulas-desenvolvedor-web
## Introdução
### O objetivo do repositório
Este repositório foi criado a fim de organizar as entregas e projetos executados no Curso de Desenvolvedor Web da CEP Escola Técnica do Guará, durante o primeiro semestre de 2026. 

### O que o projeto contém
O projeto conterá arquivos HTML, CSS e JavaScript; exercícios e outros. 

### Como executar ou visualizar os arquivos
Este projeto seguirá padrões a fim de manter a organização e facilitar a colaboração. Antes de quaisquer contribuições, atente-se às instruções que serão dispostas a seguir (em construção).
